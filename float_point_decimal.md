
#### 问题: echo floor(19.9 * 100) = 189

起初遇到这个问题的时候，总觉得是PHP内核的问题，后来使用 python 和 C 语言测试过，原来都有问题，问题抽象为：

>当对一个返回浮点数结果的乘法表达式执行隐式/显示的数值类型转换时，会出现精度丢失的可能性。

为什么是可能性呢，因为这个不是必然的，比如

```php
$fn = 1.25 * 100;
echo floor($fn);

// 输出： 125
```
